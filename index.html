dein setup — tag 1

Du brauchst genau drei Dinge um loszulegen. Nicht mehr.

Claude Code dein co-pilot

Das ist dein wichtigstes Tool. Claude Code läuft im Terminal und baut mit dir zusammen. Du beschreibst was du willst, es schreibt den Code, erklärt dir was passiert, debuggt mit dir. Nicht Copy-Paste von Stack Overflow — echtes Pair Programming mit einer AI die versteht was du meinst. Installieren: docs.anthropic.com/claude-code

VS Code editor

code.visualstudio.com — Kostenlos, läuft überall, riesiges Ökosystem. Python Extension installieren und du hast alles was du brauchst. Kein overthinking bei der Editor-Wahl — VS Code ist der Standard.

Python die einzige sprache die zählt

Python ist die Sprache der Bioinformatik, des Machine Learning und der wissenschaftlichen Datenanalyse. Alles andere kommt später oder gar nicht. Installier dir Python, öffne VS Code, lass Claude Code laufen — und bau los.

python lernen — in der richtigen reihenfolge
  • 01kaggle.com/learn — läuft im Browser, kein Setup-Stress. Python-Kurs durchmachen, dann Intro to ML. Direkt hands-on.
  • 02rosalind.info — Bioinformatik-Puzzles in Python. DNA-Sequenzen, Proteine, Algorithmen. Für Studierende gebaut. Macht süchtig.
  • 03fast.ai — Wenn du nach 2 Monaten Python kannst: hier lernst du ML praktisch. Kein Mathe-Gatekeeping, direkt bauen und verstehen.
tools die dich schneller machen

Warte nicht bis du "genug weißt". Benutz die Tools ab Tag 1 — sie sind deine Lernpartner, nicht deine Belohnung.

AlphaFold mach das heute noch

Google Colab → "AlphaFold" suchen → Proteinstruktur vorhersagen. Direkt im Browser. 10 Minuten. Kein Setup. Du siehst State-of-the-Art AI auf echte Biologie angewendet. Das ist der Moment wo es klickt.

NotebookLM papers verstehen

notebooklm.google.com — Lade Paper rein, stell Fragen, lass dir Zusammenhänge erklären. Dein persönlicher Research-Assistent. Kombinier es mit Claude: Paper in NotebookLM verstehen, dann mit Claude Code die Methoden nachbauen.

CRISPR Ressourcen dein zweites feld

CRISPR und AI verschmelzen gerade. Guide-RNA-Design wird mit ML optimiert, Off-Target-Prediction wird automatisiert. Lies The Code Breaker von Isaacson für den Einstieg, dann schau dir CRISPOR an — ein Tool das zeigt wie Computational Biology CRISPR-Design verändert.

wo du lernst

Communities und Kanäle die mehr bringen als jeder Uni-Kurs:

  • →r/bioinformatics — echte Leute, echte Probleme. Gut für Einstiegsfragen und um zu sehen was das Feld gerade bewegt.
  • →bioRxiv.org — Pre-Print-Server. Hier landen Paper bevor sie peer-reviewed sind. Bleeding edge. Erstmal über kuratierte Feeds konsumieren.
  • →StatQuest (YouTube, Josh Starmer) — der beste Erklärer für ML und Statistik der existiert. Kein Bullshit, kein Mathe-Gatekeeping.
  • →3Blue1Brown (YouTube) — Neural Networks und Lineare Algebra visuell erklärt. Das Fundament das du brauchst.
  • →Manolis Kellis / MIT OCW (YouTube) — Computational Biology Vorlesungen. Dicht aber outstanding. Für wenn du tiefer willst.
